## 本次目标
v1.3.1 目标：在 v1.3.0 基线上，继续完成电流环稳定性与切换策略收敛，再推进齿槽前馈与安全边界收尾。

### 优先级与阶段划分

执行顺序说明：先完成 P0（链路重构与快速电流环骨架），再执行 P1（电流环调优），最后执行 P2（齿槽前馈）。

定位约束：单电机 FOC 驱动库，集成调试协议与错误处理；不考虑多电机实例扩展。

### 代码现状评估（基于当前代码）
1. 调度链路现状：控制调度由 TIMER1 触发；SVPWM 插值已迁移到 PWM(TIMER0_UP) 中断；TIMER2 当前主要保留为实例层同步触发链路（驱动 PWM/采样同步）。
2. 控制频率现状：PWM 24kHz；调度 tick 为 2kHz，控制任务当前配置 1kHz，低内阻电机下仍需提升电流环有效带宽。
3. 电流环现状：已完成“外环低频 + 电流环 PWM ISR 高频”基础解耦：控制任务仅更新 iq_ref 与电角度，电流环在 PWM ISR 按分频执行。
4. 可行性结论：将插值与电流环迁移到 PWM 中断可行，但必须分层改造，禁止把 I2C 角度读取和协议处理带入高频 ISR。
5. 必要性结论：对低内阻电机，P0 为高必要任务；若不先完成 P0，P1 调参效率和稳定性都将受限，P2 齿槽前馈也缺乏稳定电流基线。

P0.2 中断梳理与PWM 插值迁移到 PWM 中断
- 目标：插值执行点从 TIMER2 回调迁移至 PWM 定时器中断，保持占空比更新与 PWM 周期同相。
- 交付：
	- PWM 中断回调通路（平台层 + 实例层 + ISR转发）完成。
	- TIMER2 仅保留同步触发主源职责，不再承载插值算法执行。
	- 验证：占空比插值连续性与周期一致性。

P0.3 快速电流环位置重构（PWM ISR）
- 目标：电流环计算频率提升至控制环的 5-10 倍，并放入 PWM 高频中断执行。
- 交付：
	- 外环与内环解耦：外环在控制调度（低频）生成 iq_ref，电流环在 PWM ISR（高频）闭环。
	- 高频路径仅使用最新电流采样缓存，不引入角度 I2C 读取。

P0.4 电流环软切换策略
- 目标：在小电流噪声区允许电流环降级，避免切换尖峰。
- 交付：
	- 开关阈值 + 回滞 + 软切换时间常数。
	- 模式切换期间 uq 输出平滑策略。
	- 验证：切换过程无明显电流冲击。
- 当前状态（2026-04-13）：
	- [x] 协议侧完成最小集/可裁剪组拆分（P/S/Y + 宏分组裁剪）。
	- [x] 软切换参数与状态通道接入控制层运行时配置。
	- [ ] 宏组合回归矩阵与实机切换冲击验证待补全。

P0.5 安全边界复核
- 目标：确保 set_voltage 限制在初始化、校准、控制、SVPWM 全链路一致生效。
- 交付：
	- 全链路限制点清单与实测记录。
	- 低内阻电机上电前的参数检查单（set_voltage、phase_resistance、目标速度/角度）。

### P1 电流环调优（当前效果差，暂时关闭）
P1.0 基础抗噪策略（强制执行）
- 目标：无论采样噪声大小，必须默认具备电流环抗噪基础能力，不依赖硬件改版。
- 交付：
	- 电流误差死区 + 积分抑制（死区内抑制积分累积，降低噪声驱动抖动）。
	- Ki 分段（低电流区降 Ki，中高电流区恢复额定 Ki）。
	- 回归验证：低电流抖动收敛且中高电流动态性能无明显退化。
- 当前状态（2026-04-13）：
	- [x] 算法实现已接入电流 PID 路径。
	- [ ] 参数整定与实验记录待补全。

### P2 齿槽前馈（Cogging Feedforward）
P2.1 学习流程落地
- 目标：建立机械角-补偿量的离线学习流程，支持重复采集与覆写。
- 交付：
	- 学习阶段执行步骤（步进角度、采样窗口、滤波/去偏置规则）。
	- 查表数据结构定义（角度分辨率、插值策略、限幅策略）。
	- 采样与应用时角度基准一致性检查项。

P2.2 运行时前馈接入

	- 运行期查表补偿路径（含插值与越界保护）。
	- 验证报告：低速纹波/抖动前后对比。
